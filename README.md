# redis-checker
Checks for exposed Redis servers

```
usage: redis-checker.py [-h] [-u URL] [-f FILE] [-p PROXY] [-P PORT] [-s SSL]

options:
  -h, --help            show this help message and exit
  -u URL, --url URL     URL to test
  -f FILE, --file FILE  File of urls
  -p PROXY, --proxy PROXY
                        Proxy for debugging
  -P PORT, --port PORT  Default Port
  -s SSL, --ssl SSL     Use SSL
  ```
